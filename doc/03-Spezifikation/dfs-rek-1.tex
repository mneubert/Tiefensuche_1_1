\section{Modul \texttt{dfs-rek-1}}
\label{sec:dfs-rek-1}

Das Modul \texttt{dfs-rek-1} beinhaltet die rekursive Implementierung der Tiefensuche und genügt der Spezifikation \texttt{digraph-dfs-proc} (siehe Abs. \ref{sec:digraph-dfs-proc}). 
\par
Die Implementierung ist orientiert sich an der in Abs. \ref{sec:implementierungen} beschreibenen Variante einer rekursiven Tiefensuche. Zur Vereinfachung des Terminationsbeweises wurde jedoch wie bei der iterativen Umsetzung zusätzlich die Hilfsfunktion \texttt{getM(.,.)} verwendet. In \texttt{decl-01} erfolgt daher zunächst der Aufruf dieser Funktion, um die Anzahl der erreichbaren Knoten vom Startknoten zu bestimmen. Danach wird die Tiefensuche gestartet, deren Quellcode \texttt{decl-03} zeigt. Nach der Intialisierung der Rückgabevariablen \texttt{bbool} mit dem Wahrheitswert \texttt{false} werden die Abbruchbedingungen geprüft. Ist weder die open-Liste noch die Zielknotenmenge leer sowie die Anzahl der noch erreichbaren Knoten größer 0, so wird der erste Knoten der open-Liste entnommen. Für diesen wird geprüft, ob er sich in der Zielknotenmenge befindet. In diesem Fall war die Suche erfolgreich und wird beendet. Anderfalls wird der Knoten der close-Liste hinzugefügt und expandiert. Zur Expansion wird die Graphenbiblioteksfunktion outsortSuccs genutzt (vgl. \cite{dr:2008}). Abschließend wird die Anzahl der erreichbaren Knoten dekrementiert und es efolgt ein rekursiver Aufruf der Funktion, sofern noch Knoten erreichbar sind.

{\LARGE\bf decl-03}

\medskip

deptSearchProc\#(open, close, z, g, n; {\bf var} bbool) \\
\{\\
bbool := false;\\
 {\bf if} \Not (open = [] \Or z = [] \Or n = 0) {\bf then}\\
 \tabif\  {\bf let} a = open.first {\bf in}\\
 \tabif\ {\bf if} a $\in$ z {\bf then} bbool := true {\bf else}\\
 \tabif\  {\bf \{}\\
 \tabif\ \tabbe close := a ' + close;\\
 \tabif\ \tabbe open := set2list(outsortSuccs(a, list2set(close), g)) + open.rest;\\
 \tabif\ \tabbe n := n - 1;\\
 \tabif\ \tabbe {\bf if} n $\neq$ 0 {\bf then} deptSearchProc\#\\
 \tabif\ {\bf \}}\\
\}

\begin{itemize}

\item Type: an axiom

\item       no proof exists
\item       used by      : Exp-01, Term-deptsearchpjsiz-01
\end{itemize}
